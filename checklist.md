## **NebuX Core Checklist**

### **Componentes**

- [ ] **Cards**
  - [ ] Definir tamaños de las tarjetas.
  - [ ] Establecer bordes (radius, color).
  - [ ] Configurar sombras para las tarjetas.
  - [ ] Establecer estilos para diferentes tipos de tarjetas (info, éxito, advertencia, etc.).
- [ ] **Inputs**
  - [ ] Estandarizar campos de texto (TextField).
  - [ ] Establecer estilos para dropdowns y selectores de fecha.
  - [ ] Implementar validación de formularios y manejo de errores.
  - [ ] Definir estilos para textos de ayuda o mensajes de error.
- [ ] **Botones**
  - [ ] Establecer el estilo para el botón primario (color, bordes, tamaño).
  - [ ] Establecer el estilo para el botón secundario.
  - [ ] Establecer el estilo para el botón de texto.
  - [ ] Definir los estados de los botones (activo, deshabilitado, cargando).
- [ ] **Shimmers**
  - [ ] Crear el efecto shimmer para estados de carga.
  - [ ] Definir color del shimmer.
  - [ ] Establecer la duración y velocidad de animación del shimmer.

---

### **Configuración Central (Theme)**

- [ ] **Colores**
  - [ ] Definir colores primarios, secundarios, de fondo, superficie, error, etc.
  - [ ] Asegurar soporte para modo claro y oscuro.
- [ ] **Tipografía**

  - [ ] Definir la familia de fuentes.
  - [ ] Establecer tamaños de fuentes (títulos, cuerpo, subtítulos).
  - [ ] Definir pesos y alturas de línea para los diferentes textos.

- [ ] **Espaciado y Distribución**
  - [ ] Establecer márgenes y rellenos estándar.
  - [ ] Definir unidades de espaciado (por ejemplo, 8px, 16px).

---

### **Extensiones**

- [ ] Crear extensiones personalizadas para Widgets.
- [ ] Crear extensiones para Cadenas (Strings).
- [ ] Definir extensiones para operaciones comunes (por ejemplo, manejo de fechas, validación de formularios).

---

### **Utilitarios (Utils)**

- [ ] Crear funciones para formateo de fechas, números, etc.
- [ ] Crear utilitarios para la validación de datos (como números de teléfono, correos electrónicos).
- [ ] Funciones para gestionar tamaños de pantalla y dispositivos.
- [ ] Métodos para manejar datos y estados de manera eficiente.

---

### **Enums**

- [ ] Definir enums para los estados de botones (activo, deshabilitado, cargando).
- [ ] Crear enums para modos de tema (claro/oscuro).
- [ ] Definir enums adicionales según sea necesario para la app (por ejemplo, tipos de notificaciones, categorías de contenido).
